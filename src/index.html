Sidewalk Priorities:

All Points of Interest

This map features a deep dive into the pedestrian accessibility of essential services in Montgomery County. These points of interest were identified through DVRPC's "Equity Through Access" study .

Each point's color reflects the relative sizes of a 1-mile walkshed using the sidewalk and OpenStreetMap networks. Green indicates that the roads immediately around a location feature an accessible sidewalk network, while red dots highlight locations where the roadway network lacks sidewalks.

Click on a single point to learn more about its network accessibility, and select a different essential service category using the drop-down list below.
